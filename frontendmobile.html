Project
Web Development

Challenge
Create a website for a non-profit that promotes responsible social media use by providing consumers with fact-based strategies to foster a more enjoyable user experience. The site must embody BotSpot’s mission to educate social media users age 65+ by providing information about the inner workings of the social media industry and encouraging them to think before they share.  

Result
I designed a visually attractive site that incorporates design elements tailored to the target market, such as a light, breathable text layout with easy-to-read font, larger buttons, and minimal need for navigation across pages.
